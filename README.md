<b>Overview</b>
This project demonstrates a complete data analysis pipeline for a fictional coffee shop, 
transforming raw transactional data into actionable insights.
It covers the full lifecycle from data extraction and transformation in Snowflake to interactive dashboards in Excel</br>

<b>Objectives</n>
Build an end-to-end ETL pipeline using Snowflake

Perform SQL-based data cleaning, aggregation, and analysis

Visualize key business metrics using Excel pivot tables and dashboards

Support decision-making for inventory, sales trends, and customer behavior</br>

<b>Tools & Technologies</b>
Tool&Purpose
Snowflake-Cloud-based data warehousing & ETL
SQL-Data querying, transformation & joins
Excel-Pivot tables, charts, and dashboards
CSV/Flat-Files	Source data ingestion</br>
<b>ETL Workflow</b>
Extract-Load raw sales and inventory data from CSV files into Snowflake staging tables.
Transform:
Clean and normalize data using SQL
Join datasets (e.g., sales + products + employees)
Create views for reporting
Load: Export transformed data to Excel for visualization
